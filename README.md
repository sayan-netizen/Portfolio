# Sayan Mondal | Developer Portfolio

Welcome to the repository for my personal developer portfolio website. This site serves as a central hub for my professional experience, academic background, technical skills, and featured case studies.

## 🚀 Overview

This portfolio is built with a focus on delivering a high-performance, accessible, and visually striking digital experience. The design features a modern **Neo-Brutalist** aesthetic combined with **Glassmorphism** concepts, unified under a sleek dark mode theme.

## 🛠️ Technologies Used

- **HTML5:** Semantic and accessible document structure.
- **CSS3:** Custom styling utilizing CSS variables, Flexbox/Grid layouts, hard-shadowed brutalist UI components, and backdrop-filter glass overlays.
- **JavaScript (Vanilla):** Interactive elements, active navigation tracking, and smooth scroll behaviors.

## ✨ Features

- **Responsive Design:** Fully fluid layout that adapts seamlessly from ultra-wide desktops to mobile devices.
- **Dynamic Aesthetic:** A unique blend of stark Neo-Brutalist borders/shadows and translucent Glassmorphism elements.
- **Project Showcase:** Detailed highlight cards for personal and academic projects, including:
  - Binance Futures Testnet Trading Bot (Python, REST API)
  - E-Nose Perfume Classification (IoT, Machine Learning)
  - Multi-Layer Honeypot System (Cybersecurity, Docker)
- **Interactive Timeline:** A structured layout for my Web Development Internship at ApexPlanet and academics at UEM Kolkata.
- **Contact Form:** Integrated section for direct inquiries.

## 💻 Running Locally

Since this is a static website, getting it up and running locally is extremely simple:

1. Clone this repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open the `index.html` file in your preferred web browser, or use an extension like **VS Code Live Server** to launch a local development server for the best experience.
